# [[患]] *[[huàn]]*  [[uuid-ccee9f93-d493-43f0-b41f-64aa72876a47|vtoS]] [[PROBLEM]]
run into the problem that S, be faced with the problem that
 - [[陸機集十一卷]] 恆**患**意不稱物，
                     / 患 "run into the problem" is not fear of something but exposure to a problem/danger.  *意 is not philosophical "ideas men have" but autobiographic: "my ideas". *稱 refers to correspondence: "correspond properly, capture properly", not equality or balance. *物 refers not to things generally but to the things dealt with by one's 意 "ideas/notions".[HXWD](https://hxwd.org/textview.html?location=CH2b1575_CHANT_001-1a.11)
