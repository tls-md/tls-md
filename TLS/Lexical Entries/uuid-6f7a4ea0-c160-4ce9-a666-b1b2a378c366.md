# [[尼師壇]] *[[ní shī tán]]*  [[uuid-a8e89bab-49e1-4426-b230-0ec7887fd8b4|NP]] [[MAT]]
soft mat used by Buddhists DCD: 亦作尼師但那。梵語音譯。佛教徒所說六物之一。即隨坐衣。衣上所附布料可作坐具或臥具。
