# Lexical Entities
- (pinyin::*shí xiàng wú xiàng*) [[TLS/Core/Syntactic functions/uuid-155b3f78-76bb-4fc8-9f02-6d3f28b5d00d|(syn-func::{VP...})]] (def::**BUDDH: the true mark which has no features, the true mark is No-mark (see 實相 and 無相) (this is a very popular phrase in Chán/Zen texts, expressing that the true nature of things is without any distinguishing features, and as such the true nature of all things is identical; the phrase is originally based on the DAZHI DULUN, see SOURCES; see also Wúliàng yì jīng 無量義經: “其一法者即無相也，如是無相，無相不相，不相無相，名為實相。”(T.9: 385c).
