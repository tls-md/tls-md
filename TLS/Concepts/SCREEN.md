# Definition
THIN WALL which CAN be MOVED, OR WALL WHICH is USED to BLOCK the WIND OR LOOKING.
# Criteria and general notes
## 
HUANG JINGUI 2006
ARCHITECTURE 14. 宮室地面障蔽物。
屏，上古是門屏即門內外土牆垣總稱。
樹，屏的別稱。蕭牆，外屏。
扆，人君專用的禮儀性小型木屏。
罘罳，漢代雕鏤有網格紋的屏狀木建築，見於門闕之外、高層建築之間連接的連閣、角樓；其木構部分所雕鏤網格紋為主的華飾也稱 “ 罘罳 ” 。
屏風，戰國起流行，漢魏後屏風或稱 “ 屏 ” 。
